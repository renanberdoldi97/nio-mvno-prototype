# Nio MVNO Prototype

Protótipo interativo do lançamento do chip móvel Nio (MVNO como benefício da fibra).
Uso interno — apresentação a stakeholders e diretoria.

## Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS v4
- Framer Motion
- Zustand para state machine

## Rodando localmente
npm install
npm run dev

Acesse http://localhost:3000 — otimizado para mobile (viewport ≤ 430px).

## Assets
Os ícones e logos do DS Nio já estão copiados em:
- /public/icons/ ← ícones do DS Nio
- /public/logo/ ← 3 SVGs de logo (Default, White, Highlight)
